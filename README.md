# zsh-uv

A zsh plugin for the [uv cli](https://github.com/astral-sh/uv).

## Install
Using the [:zap: Zap](https://www.zapzsh.org/) minimal zsh plugin manager, add the following to your `.zshrc`

```sh
# Install plugin
plug "lipov3cz3k/zsh-uv"
```
